How To Create a Project Using MvcCsharpTemplate2020
====

1. Create a copy of the folder MvcCsharpTemplate2020
2. Rename the folder copy of MvcCsharpTemplate2020 to your project name (Example: MvcProject1)
3. Inside the folder MvcProject1:
	a. Rename MvcCsharpTemplate2020.sln to MvcProject1.sln
	b. Rename folder MvcCsharpTemplate2020 to MvcProject1
4. Edit MvcProject1.sln in text editor, and change all instances of "MvcCsharpTemplate2020" to "MvcProject1"
5. Inside folder MvcProject1\MvcProject1:
	a. Rename MvcCsharpTemplate2020.csproj to MvcProject1.csproj
	b. Rename MvcCsharpTemplate2020.csproj.user to MvcProject1.csproj.user
6. Edit MvcProject1.csproj in text editor, and change all instances of "MvcCsharpTemplate2020" to "MvcProject1"
7. Open MvcProject1.sln in text editor Solution in Visual Studio.
8. On the Solution Explorer, Open any controller in the Controllers Folder.
	a. On the namespace code line select and highlight the text "MvcCsharpTemplate2020"
	b. Press CTRL + H and rename all occurance of "MvcCsharpTemplate2020 with "MvcProject1" on the Current Project.
9. Test by Rebuilding Project.
